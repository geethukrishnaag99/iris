This project predicts the species of an Iris flower using a Support Vector Machine (SVM).

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Flask
- NumPy

## How to Run

### 1. Install requirements
pip install -r requirements.txt

### 2. Train the model
python train_and_save_model.py

### 3. Run Streamlit app
streamlit run iris.py

### 4. Run Flask app
python app.py
